# JuanDiegoSoto_Katas_Python
 DataProject: Lógica. Katas Python - Juan Diego Soto

## 📖 Descripción

Este proyecto contiene la resolución completa de 41 ejercicios (katas) de programación en Python diseñados para practicar y dominar conceptos fundamentales del lenguaje. El objetivo es desarrollar habilidades en programación funcional, manejo de excepciones, programación orientada a objetos y estructuras de datos.

Los ejercicios abarcan desde operaciones básicas con strings y listas hasta implementación de clases complejas y manejo avanzado de errores, utilizando técnicas modernas de Python como `map()`, `filter()`, `reduce()`, funciones lambda y programación orientada a objetos.

## 🛠 Instalación y Requisitos

Este proyecto usa **Python 3.13.12** y requiere las siguientes bibliotecas estándar:

- `functools` (incluida en Python)
- `typing` (incluida en Python)
- `math` (incluida en Python)

### Instalación

1. Clona o descarga el repositorio
2. Asegúrate de tener Python 3.8 o superior instalado
3. Ejecuta el archivo de soluciones:

```bash
JuanDiegoSoto_katas_python_soluciones.py
```

No se requiere instalación de dependencias externas, ya que todas las bibliotecas utilizadas son parte de la librería estándar de Python.

## 📚 Contenido de los Ejercicios

### Nivel Básico (1-15)
- Manipulación de cadenas y diccionarios
- Uso de funciones `map()`, `filter()` y `reduce()`
- Recursividad y funciones lambda
- Manejo básico de excepciones

### Nivel Intermedio (16-30)
- Procesamiento avanzado de listas
- Validación de datos con excepciones personalizadas
- Algoritmos de búsqueda y transformación
- Operaciones con funciones de orden superior

### Nivel Avanzado (31-41)
- Programación orientada a objetos (POO)
- Clases `Arbol` y `UsuarioBanco` con métodos complejos
- Procesamiento de texto con múltiples estrategias
- Programas interactivos con validación robusta
